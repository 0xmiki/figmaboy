---
title: Keyboard shortcuts
description: Figmaboy canvas, selection, layer, history, zoom, panel, and terminal shortcuts.
---

Use <kbd>Cmd</kbd> instead of <kbd>Ctrl</kbd> for standard modifier shortcuts on macOS. The terminal toggle is always <kbd>Ctrl</kbd> + <kbd>`</kbd>.

## Tools

| Action | Shortcut |
| --- | --- |
| Select / move | <kbd>V</kbd> |
| Hand / pan | <kbd>H</kbd> |
| Frame | <kbd>F</kbd> or <kbd>A</kbd> |
| Rectangle | <kbd>R</kbd> |
| Ellipse | <kbd>O</kbd> |
| Line | <kbd>L</kbd> |
| Arrow | <kbd>Shift</kbd> + <kbd>L</kbd> |
| Text | <kbd>T</kbd> |
| Toggle embedded terminal | <kbd>Ctrl</kbd> + <kbd>`</kbd> |

## Selection and editing

| Action | Shortcut |
| --- | --- |
| Select all | <kbd>Ctrl/Cmd</kbd> + <kbd>A</kbd> |
| Copy / cut / paste | <kbd>Ctrl/Cmd</kbd> + <kbd>C/X/V</kbd> |
| Duplicate | <kbd>Ctrl/Cmd</kbd> + <kbd>D</kbd> |
| Delete | <kbd>Delete</kbd> or <kbd>Backspace</kbd> |
| Nudge 1 px | Arrow keys |
| Nudge 10 px | <kbd>Shift</kbd> + arrow keys |
| Select first child | <kbd>Enter</kbd> |
| Select parent | <kbd>Shift</kbd> + <kbd>Enter</kbd> |
| Cancel / return to select | <kbd>Esc</kbd> |

## Group and arrange

| Action | Shortcut |
| --- | --- |
| Group | <kbd>Ctrl/Cmd</kbd> + <kbd>G</kbd> |
| Ungroup | <kbd>Ctrl/Cmd</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd> |
| Frame selection | <kbd>Ctrl/Cmd</kbd> + <kbd>Alt</kbd> + <kbd>G</kbd> |
| Toggle visibility | <kbd>Ctrl/Cmd</kbd> + <kbd>Shift</kbd> + <kbd>H</kbd> |
| Toggle lock | <kbd>Ctrl/Cmd</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd> |
| Bring forward / to front | <kbd>]</kbd> / modified <kbd>]</kbd> |
| Send backward / to back | <kbd>[</kbd> / modified <kbd>[</kbd> |

## History, zoom, and panels

| Action | Shortcut |
| --- | --- |
| Undo | <kbd>Ctrl/Cmd</kbd> + <kbd>Z</kbd> |
| Redo | <kbd>Ctrl/Cmd</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd> or <kbd>Ctrl/Cmd</kbd> + <kbd>Y</kbd> |
| Zoom in | <kbd>+</kbd> or <kbd>Ctrl/Cmd</kbd> + <kbd>=</kbd> |
| Zoom out | <kbd>Shift</kbd> + <kbd>-</kbd> or <kbd>Ctrl/Cmd</kbd> + <kbd>-</kbd> |
| Fit all | <kbd>Shift</kbd> + <kbd>1</kbd> |
| Fit selection | <kbd>Shift</kbd> + <kbd>2</kbd> |
| Reset to 100% | <kbd>Shift</kbd> + <kbd>0</kbd> |
| Toggle side panels | <kbd>Ctrl/Cmd</kbd> + <kbd>\</kbd> or <kbd>Ctrl/Cmd</kbd> + <kbd>.</kbd> |

Trackpad or mouse-wheel input pans the canvas. Hold <kbd>Ctrl</kbd> / <kbd>Cmd</kbd> while scrolling to zoom around the pointer.
